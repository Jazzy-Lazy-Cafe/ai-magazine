# Podcast Summary 품질 체크리스트

## 1. 유기적 연결성 (30점)

### 1.1 Opening → Body 연결 (10점)
| 점수 | 기준 |
|------|------|
| 10점 | opening_insight의 핵심 주제가 첫 2-3개 섹션에서 깊이 있게 탐구됨 |
| 7점 | opening_insight와 초반 섹션의 연결이 있지만 직접적이지 않음 |
| 4점 | opening_insight가 본문에서 한 번 정도만 언급됨 |
| 0점 | opening_insight와 본문 사이에 논리적 연결이 없음 |

**체크 방법**:
- opening_insight의 핵심 키워드를 추출
- sections[0-2]에서 해당 키워드가 발전되는지 확인
- 단순 반복이 아닌 "확장/구체화/반박" 등의 발전이 있는지 확인

### 1.2 섹션 간 흐름 (15점)
| 점수 | 기준 |
|------|------|
| 15점 | 모든 섹션 전환이 자연스럽고, 독자가 "왜 이 질문?"을 묻지 않음 |
| 10점 | 대부분의 전환이 자연스럽지만 1-2개의 갑작스러운 전환 존재 |
| 5점 | 절반 정도의 전환만 자연스러움 |
| 0점 | 섹션들이 독립적으로 나열되어 있고 흐름이 없음 |

**체크 방법**: 각 섹션 쌍(N, N+1)에 대해:
- [ ] N의 answer 마지막 문장이 N+1의 question과 논리적으로 연결되는가?
- [ ] 연결 패턴 식별 가능한가? (키워드/인과/시간순/구체화)

### 1.3 Body → Footer 수렴 (5점)
| 점수 | 기준 |
|------|------|
| 5점 | footer가 전체 논의를 완벽하게 마무리하고 핵심 메시지를 전달 |
| 3점 | footer가 관련은 있지만 전체를 수렴하지는 못함 |
| 0점 | footer가 본문과 동떨어진 인용문 사용 |

## 2. 독자 친화성 (30점)

### 2.1 두괄식 답변 (10점)
| 점수 | 기준 |
|------|------|
| 10점 | 모든 answer가 질문에 대한 직접 답변으로 시작 |
| 7점 | 80% 이상의 answer가 두괄식 |
| 4점 | 50-79%의 answer가 두괄식 |
| 0점 | 대부분의 answer가 배경 설명으로 시작 |

**체크 방법**: 각 answer의 첫 문장 분석
- 좋음: "AI는 지수적으로 발전하고 있습니다."
- 나쁨: "이 질문에 답하기 위해서는 먼저 배경을 이해해야 합니다."

### 2.2 전문용어 처리 (10점)
| 점수 | 기준 |
|------|------|
| 10점 | 모든 전문용어가 knowledge_items에 명확하게 설명됨 |
| 7점 | 대부분의 전문용어가 설명되었지만 1-2개 누락 |
| 4점 | 절반의 전문용어만 설명됨 |
| 0점 | 전문용어 설명이 거의 없거나 설명 자체가 어려움 |

**체크 방법**:
- answer에서 전문용어 추출
- knowledge_items에서 해당 용어 검색
- 설명이 "~이란 ~입니다" 형태로 명확한지 확인

### 2.3 구체성 (10점)
| 점수 | 기준 |
|------|------|
| 10점 | 모든 answer에 숫자/비유/사례 중 최소 1개 포함 |
| 7점 | 80% 이상의 answer에 구체적 요소 포함 |
| 4점 | 절반 정도의 answer만 구체적 |
| 0점 | 대부분 추상적 서술 |

**구체성 요소 예시**:
- 숫자: "2년간 100배 성장", "5-10% GDP 성장"
- 비유: "MRI가 뇌를 보듯이"
- 사례: "Claude Opus 4.5를 사용하면서..."

### 2.4 자연스러운 한국어 (감점 요소)

번역투가 심할 경우 독자 친화성 점수에서 최대 -5점 감점

| 감점 | 기준 |
|------|------|
| 0점 | 자연스러운 한국어, 번역투 없음 |
| -2점 | 1-2개의 어색한 번역투 표현 존재 |
| -5점 | 다수의 번역투 표현으로 가독성 저하 |

**번역투 체크 항목**:
- [ ] "~자(者)" 형태의 어색한 조어가 없는가? (사고자, 수행자 등)
- [ ] "그것은", "이것은"으로 시작하는 문장이 적은가?
- [ ] 영어식 강조 구문("~하는 것은 ~이다")이 없는가?
- [ ] 불필요하게 긴 수동태 표현이 없는가?
- [ ] 문장이 지나치게 길지 않은가?

**흔한 번역투와 자연스러운 대안**:

| 어색한 번역투 | 자연스러운 표현 |
|---------------|-----------------|
| 사고자, 생각자 | 생각이 느린 사람 |
| ~하는 것은 가능하다 | ~할 수 있다 |
| ~에 대해 흥분하고 있다 | ~이 기대된다 |
| 그것은 ~이다 | (주어 생략) ~이다 |
| ~라는 사실 | ~다는 점 |
| ~를 수행하다 | ~를 하다 |
| ~하는 경향이 있다 | ~하곤 한다 |

**예시**:
- 어색: "당신은 느린 사고자일 뿐이에요"
- 자연스러움: "당신은 생각이 느릴 뿐이에요"

| 점수 | 기준 |
|------|------|
| 15점 | 인터뷰의 핵심 주장과 새로운 통찰이 모두 포함됨 |
| 10점 | 핵심 내용 포함되었으나 일부 중요 포인트 누락 |
| 5점 | 표면적 내용만 포함, 깊이 있는 인사이트 부족 |

|------|------|

| 3점 | 1-2개의 answer만 범위 초과/미달 |
| 0점 | 3개 이상의 answer가 범위 벗어남 |

| 점수 | 기준 |
|------|------|
