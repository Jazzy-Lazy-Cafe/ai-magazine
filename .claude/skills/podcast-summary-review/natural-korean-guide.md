1. 명사 중심 구문을 동사/형용사로 (1~20)
   영어가 선호하는 명사 중심 표현을 한국어다운 서술어 중심으로 고친 사례입니다.
   관심을 가지다 → 관심을 두다 / 궁금해하다
   어려움이 있다 → 어렵다
   확인을 하다 → 확인하다
   준비가 되다 → 준비하다
   결정을 내리다 → 결정하다
   생각을 가지고 있다 → 생각하다
   기대를 모으다 → 기대하다
   싸움을 벌이다 → 싸우다
   사랑을 하다 → 사랑하다
   휴식을 취하다 → 쉬다
   조치를 취하다 → 대책을 세우다
   이동을 시키다 → 옮기다
   감동을 주다 → 감동적이다
   영향을 미치다 → 영향을 주다 / 변하게 하다
   도움을 주다 → 돕다
   거절을 당하다 → 거절당하다
   발표를 가지다 → 발표하다
   미소를 짓다 → 웃다
   질문을 던지다 → 묻다
   실수를 저지르다 → 실수하다

2. 수동태(피동형)를 능동형으로 (21~40)
   영어의 수동태를 한국어의 자연스러운 능동태로 바꾼 사례입니다.
   ~에 의해 작성되다 → ~가 쓰다
   ~라 불리다 → ~라고 하다
   ~라 생각되어지다 → ~라고 보다/생각하다
   보여지다 → 보이다
   쓰여지다 → 쓰이다
   그에 의해 발견되다 → 그가 발견하다
   도둑에 의해 지갑을 털리다 → 도둑이 지갑을 털어가다
   이용되어지다 → 이용되다 / 쓰다
   열려지다 → 열리다
   믿겨지다 → 믿기다
   나뉘어지다 → 나뉘다
   잊혀지다 → 잊히다
   찢겨지다 → 찢기다
   함유되어 있다 → 들어 있다
   배치되어 있다 → 놓여 있다
   설계되다 → 만들다 / 설계하다
   제한되다 → 줄이다 / 막다
   요구되다 → 필요하다
   추천되다 → 권하다
   확인되어지다 → 확인되다

3. 무생물 주어 구문 순화 (41~60)
   사물이 사람처럼 행동하는 표현을 원인/이유나 부사구로 고친 사례입니다.
   이 약은 통증을 멈추게 한다 → 이 약을 먹으면 통증이 멎는다
   무엇이 당신을 여기 오게 했나요? → 여기 왜 오셨나요?
   태풍이 기차를 멈추게 했다 → 태풍 때문에 기차가 멈췄다
   그의 말이 나를 화나게 했다 → 그 말을 듣고 화가 났다
   세월은 우리를 변하게 한다 → 세월이 흐르면 우리도 변한다
   이 길은 학교로 통한다 → 이 길로 가면 학교가 나온다
   그 뉴스는 나를 놀라게 했다 → 그 뉴스를 듣고 놀랐다
   바쁜 업무가 나를 힘들게 한다 → 업무가 바빠서 힘들다
   비가 내 마음을 슬프게 한다 → 비가 오니 마음이 슬프다
   음악이 방 안을 채웠다 → 방 안에 음악이 울려 퍼졌다
   사고가 그의 인생을 바꿨다 → 사고로 그의 인생이 바뀌었다
   증거가 유죄를 입증한다 → 증거를 보니 유죄가 확실하다
   신문이 말하길 → 신문에 따르면
   그의 표정이 거절을 의미했다 → 그는 거절하듯 표정을 지었다
   경험이 그를 성장시켰다 → 그는 경험을 통해 성장했다
   이 요리는 입맛을 돋운다 → 이 요리를 먹으면 입맛이 산다
   알람이 나를 깨웠다 → 알람 소리에 잠이 깼다
   거울이 내 모습을 비춘다 → 거울에 내 모습이 비친다
   술이 그를 취하게 했다 → 그는 술에 취했다
   편지가 소식을 전했다 → 편지로 소식을 들었다

4. 불필요한 조사 및 지시어 제거 (61~80)
   '~의', '그/그녀', '복수형' 등 불필요한 표현을 다듬은 사례입니다.
   나의 사랑의 노래 → 내가 사랑하는 노래 / 내 사랑 노래
   그와 그녀 → (이름 지칭 또는 생략)
   많은 사람들 → 사람도 많다 / 많은 사람
   사과들 → 사과 (한국어는 문맥상 복수를 생략함)
   그것들 → (구체적인 사물 이름)
   우리들의 학교 → 우리 학교
   서로에 대한 사랑 → 서로 아끼는 마음
   ~로의 초대 → ~에 초대함
   성공으로의 길 → 성공하는 길
   최선의 선택 → 가장 좋은 선택
   미래에의 꿈 → 미래를 향한 꿈
   모두에 의한 참여 → 모두가 참여함
   친구와의 만남 → 친구를 만남
   하나의 예 → 예 한 가지
   그 자신의 생각 → 제 생각 / 자기 생각
   당신의 이름 → 이름 (공손한 표현 시 주어 생략)
   그들만의 리그 → 그들끼리의 잔치
   책들 속의 지식 → 책 속의 지식
   오늘의 뉴스 → 오늘 뉴스
   국가 간의 분쟁 → 국가 간 분쟁

5. 상투적 번역구 및 숙어 표현 (81~100)
   영어 관용구를 그대로 직역한 표현을 고친 사례입니다.
   ~중의 하나 → ~인 하나 / ~가운데 하나
   ~에 관하여 → ~을/를 / ~에 대해
   ~함에 있어서 → ~할 때
   ~에도 불구하고 → ~인데도 / ~하지만
   ~와/과 함께 → ~와/과 / ~를 데리고
   ~를 필요로 하다 → ~이 필요하다
   ~할 가치가 있다 → ~할 만하다
   ~하기 위하여 → ~하려고 / ~하러
   ~에 직면하다 → ~에 맞닥뜨리다
   ~할 예정이다 → ~할 것이다 / ~하기로 했다
   ~을 바탕으로 → ~에 따라 / ~을 토대로
   ~을 특징으로 하다 → ~이 특징이다
   ~을 제외하고 → ~말고는 / ~외에
   ~로부터 → ~에게서 / ~에서
   아무리 강조해도 지나치지 않다 → 매우 중요하다
   좋은 시간을 보내다 → 즐겁게 지내다
   기회를 가지다 → 기회를 얻다 / 기회가 생기다
   ~할 준비가 되다 → ~하려고 하다
   ~라 간주되다 → ~로 보다
   최선을 다하다 → 힘껏 하다 / 온 힘을 쏟다
